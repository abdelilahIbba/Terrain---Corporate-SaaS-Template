File Structure

The template follows a organized file structure for easy customization:

my-saas-template/
├── index.html
├── pages/
│   ├── about.html
│   ├── pricing.html
│   └── contact.html
├── assets/
│   ├── css/
│   │   ├── style.css
│   │   └── style.min.css
│   ├── js/
│   │   ├── main.js
│   │   └── main.min.js
│   ├── img/
│   └── fonts/
├── documentation/
│   └── index.html
└── README.md

Key Files:

  • index.html - Main homepage
  • pages/ - Additional pages (about, pricing, contact)
  • assets/css/style.css - Main stylesheet with custom CSS
  • assets/js/main.js - Custom JavaScript functionality
  • assets/img/ - Directory for all images

Customization Guide

Updating Content

To update text content, simply edit the HTML files and replace the placeholder text with your actual content.

Changing Images

Replace the placeholder images in the assets/img/ directory with your own images. Update the image paths in the HTML files accordingly.

Modifying Colors

The template uses CSS custom properties for easy color customization. Edit the values in the :root section of assets/css/style.css:

:root {
    --sand-beige: #F5F5DC;
    --terracotta: #E2725B;
    --sage-green: #9CAF88;
    --deep-brown: #3E2723;
    --light-cream: #FDFDF6;
    --primary-soft: rgba(157, 175, 136, 0.1);
}

Typography

The template uses three Google Fonts for a professional yet distinctive look:

  • Poppins - Used for headings
  • Lato - Used for body text
  • Playfair Display - Used for accents and logo

To change fonts, update the Google Fonts link in the <head> section and modify the CSS font-family properties.
